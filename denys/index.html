Interactive Elements

In-text link

CSS selector:

.text-link

Sample code:

<a href="#" class="text-link">Text Link</a>

Rendered element:

Text Link

Submit/Reset Buttons

CSS selector:

.btn

Sample code:

<button href="#" class="btn">Default Button</button>

Rendered element:

CSS selector:

.btn .btn-primary

Sample code:

<button href="#" class="btn btn-primary">Primary Button</button>

Rendered element:

CSS selector:

.btn .btn-secondary

Sample code:

<button href="#" class="btn btn-secondary">Secondary Button</button>

Rendered element:

CSS selector:

.btn .btn-outline

Sample code:

<button href="#" class="btn btn-outline">Outlined Button</button>

Rendered element:

Text Input Field with Label

CSS selector:

.input

Sample code:

<div href="#" class="input">
 <label for="input">Label</label>
 <input id="input" />
</div>

Rendered element:

Supporting text

Radio Button with Label

CSS selector:

.radio

Sample code:

<div href="#" class="radio">
 <input type="radio" id="radio" />
 <label for="radio">Label</label>
</div>

Rendered element:
